          Conectar Claude significa pegar tu propia API key (se genera en la consola de tu cuenta: platform.claude.com → API Keys); no existe un «iniciar sesión con Claude» para apps de terceros. La key vive SOLO en este dispositivo (LocalStorage) y jamás se envía a ningún servidor nuestro. Habilita pistas dinámicas y variantes frescas; sin key, la app funciona completa con las pistas curadas incluidas.
