          Potenciar con Claude (opcional) — Cuentas

          Conectar tu cuenta toma 2 minutos: 1) abre la consola de Claude e inicia sesión ahí con tu correo (Gmail incluido) — platform.claude.com → API Keys —, 2) crea una API key y cópiala, 3) pégala aquí abajo. Anthropic no permite a apps de terceros un botón «iniciar sesión con Claude»; la key es la vía oficial y vive solo en este dispositivo.

            Conectar Claude significa pegar tu propia API key (se genera en la consola de tu cuenta: platform.claude.com → API Keys); no existe un «iniciar sesión con Claude» para apps de terceros. La key vive SOLO en este dispositivo (LocalStorage) y jamás se envía a ningún servidor nuestro. Habilita pistas dinámicas y variantes frescas; sin key, la app funciona completa con las pistas curadas incluidas.

            Mentor local (opcional) — RAG sobre tu biblioteca

            Si tienes el backend del mentor corriendo (RAG + DeepSeek en una laptop, ver mentor-backend/), conéctalo aquí. En el Modo Estudio, el mentor explicará apoyándose en la biblioteca. Es opcional: si está apagado, la app sigue igual. Tu API key de Claude jamás se envía a este backend.

            La URL y el token viven SOLO en este dispositivo (LocalStorage, fuera de la sincronización). Para usar el mentor local necesitas tu cuenta iniciada: el backend valida tu sesión.
